Hello Everyone. This is my project on Currency classification system using convolution neural network. In this project I have taken 6 classes whic are 10rs, 20 rs, 50rs, 100rs, 200rs, 500rs each with 3000 images 
per folder . Here I haven't use any transfer learning or pretrained models. I just build the cnn architecture from scratch and received accuracy above 90% with no underfitting and overfitting
