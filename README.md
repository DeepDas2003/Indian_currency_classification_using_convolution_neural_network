Hello Everyone. This is my project on Indian currency classification system using Convolution neural network. I have taken 6 classes which are 10rs, 20rs 50rs, 100rs, 200rs, 500rs where each folder consists of 3000 images. 
Here the main thing is that i haven't used any pretrained model for training. I simply built the cnn architecture by my hand .
